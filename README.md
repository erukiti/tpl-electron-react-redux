# Electron + React + Redux boilerplate

* Electron + React + Redux のボイラープレートです。
* アイコン加工は[Yosemite Icon Kakeru Yatsu](http://ls8h.com/yosemite-icon/)です。

# 動作

* `npm start` で開発用で起動します。
* `npm test` でテストします。
* `npm run build` で配布用パッケージをビルドします。
